Transforming - #pricelessnomore Example




Transitioning - Divs

Width - growing over 2 seconds
.div1 { background-color: red; transition-property: width; transition-duration: 2s; }
.div1:hover { width: 300px; }


Height - growing over 3 seconds
.div2 { background-color: blue; transition: height 4s; }
.div2:hover { height: 300px; }


Height and Width - Growing Height over 4 seconds and Width over 1 seconds
.div3 { background-color: green; transition: height 4s, width 1s; }
.div3:hover { height: 300px; width: 400px; }


Delay - Width grows after hovering for 1 second.
.div4 { background-color: black; transition: width 5s; transition-delay: 1s; }
.div4:hover { height: 300px; width: 400px; }


Timing function - sets the transition to end slowly.
.div5 { background-color: hotpink; transition: height 4s, width 1s; transition-timing-function: ease-out; }
.div5:hover { height: 300px; width: 400px; }
